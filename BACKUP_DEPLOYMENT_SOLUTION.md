1. **Check Build Logs**: Look for React/Vite errors
2. **Environment Variables**: Add `VITE_API_URL`
3. **Delete & Reimport**: Remove project, import fresh from GitHub

### Step 3: Alternative Deployment (If Vercel Fails)
Deploy to **Netlify** as backup:
1. Go to: https://netlify.com
2. Sign in with GitHub
3. Click: "New site from Git"
4. Select: school-management-system repository
5. Build settings:
   - Build command: `npm run build`
   - Publish directory: `dist`
