## 🚀 After GitHub Push: Deploy in 15 Minutes

Once pushed to GitHub, follow these steps:

### Step 1: Deploy Backend on Render (10 minutes)
1. Go to https://render.com
2. Sign up/Sign in with GitHub
3. Click "New +" → "Web Service"
4. Select your repository: `school-management-system`
5. Configure:
   - Root Directory: `backend`
   - Build Command: `npm install`
   - Start Command: `node index.js`
6. Add environment variables (see DEPLOYMENT_READY.md)
7. Click "Create Web Service"
8. Wait for deployment
9. Your backend will be live!

### Step 2: Deploy Frontend on Vercel (5 minutes)
1. Go to https://vercel.com
2. Sign up/Sign in with GitHub
3. Click "Add New" → "Project"
4. Import your repository
5. Configure:
   - Root Directory: `frontend`
   - Framework: Vite
   - Build Command: `npm run build`
   - Output Directory: `dist`
6. Add environment variable:
   - `VITE_API_URL` = your Render backend URL
7. Click "Deploy"
8. Wait for deployment
9. Your frontend will be live!

### Step 3: Test Everything (2 minutes)
1. Visit your Vercel URL
2. Login with admin credentials
3. Test all features
4. Done! 🎉
