🆘 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════════════════

If backend shows ECONNREFUSED error:
  1. Go to Render Environment Variables
  2. Verify all 8 variables are there
  3. Check DB_HOST is exactly: ${{RAILWAY_PRIVATE_DOMAIN}}
  4. Click Redeploy again
  5. Wait 5 minutes

If frontend shows "Failed to fetch" error:
  1. Go to Vercel Environment Variables
  2. Verify VITE_API_URL is set
  3. Click Redeploy
  4. Hard refresh browser (Ctrl+Shift+R)
  5. Try logging in again

If still not working:
  1. Wait 5 minutes (deployment takes time)
  2. Check Render logs for error messages
  3. Check browser console (F12) for errors
  4. Verify Railway database is running
