WHAT THIS ERROR MEANS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ECONNREFUSED = Connection Refused
127.0.0.1:3306 = localhost port 3306 (where MySQL runs)

This means:
- Backend tried to connect to MySQL
- But MySQL is not running
- So the connection was refused

Solution: Start MySQL in XAMPP
