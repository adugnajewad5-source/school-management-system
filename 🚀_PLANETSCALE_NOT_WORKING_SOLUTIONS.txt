PlanetScale. It's free and works great for testing.

2. Create database: school_management
3. Import schema
4. Set credentials on Render
5. Redeploy

Once you have database credentials (from any option):

1. Go to: https://dashboard.render.com
2. Click: school-management-backend-gnav
3. Click: "Environment" tab
4. Add these variables:
   - DB_HOST = (your database host)
   - DB_USER = (your database user)
   - DB_PASSWORD = (your database password)
   - DB_NAME = school_management
   - NODE_ENV = production
5. Click: "Redeploy"
6. Wait 2-3 minutes
7. Check logs for: "Connected to MySQL database"
8. Test login at: https://school-management-system.vercel.app
