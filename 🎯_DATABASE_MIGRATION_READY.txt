When backend starts on Render:

1. backend/index.js loads
2. Connects to Railway database
3. Automatically runs backend/run-railway-migration.js
4. Migration script:
   - Reads database/schema.sql
   - Creates all tables (users, students, teachers, payments, results, attendance)
   - Creates admin user with password: Admin@123
5. Backend is ready to accept requests

This happens automatically every time the backend starts!
