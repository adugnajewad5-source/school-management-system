STEP 1: Go to Render Dashboard
  → https://dashboard.render.com

STEP 2: Click Your Backend Service
  → Click: school-management-backend-gnav

STEP 3: Click Environment Tab
  → Click: "Environment" (left sidebar)

STEP 4: Remove Old Variables
  → Find and DELETE: DB_HOST=mysql.railway.internal
  → Delete any other Railway variables

STEP 5: Add New Variables
  → Click: "Add Environment Variable"
  → Add these 7 variables:

     1. DB_HOST = localhost
     2. DB_USER = root
     3. DB_PASSWORD = (leave empty)
     4. DB_NAME = school_management
     5. NODE_ENV = production
     6. JWT_SECRET = your_super_secret_jwt_key_123_change_in_production
     7. FRONTEND_URL = https://school-management-system.vercel.app

STEP 6: Redeploy
  → Click: "Redeploy" button (top right)
  → Wait: 2-3 minutes

STEP 7: Check Logs
  → Click: "Logs" tab
  → Look for: "Connected to MySQL database"
  → If you see this, it's working! ✅

STEP 8: Test Frontend
  → Open: https://school-management-system.vercel.app
  → Login with:
     Username: admin
     Password: Admin@123
  → You should see admin dashboard
